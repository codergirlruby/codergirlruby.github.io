Atom Over Sublime

Github sent me a really neat virtual backpack with a bunch of fun goodies inside.

such as:

Atom — A hackable text editor for the 21st Century Bitnami — Install cloud applications in a single click Crowdflower — Crowdsourcing and data enrichment platform DigitalOcean — Simple cloud hosting, built for developers DNSimple — Simple DNS management with one-click services and a robust API GitHub — Powerful collaboration, code review, and code management HackHands — Live programming help available 24/7 Namecheap — Domain name registration and SSL certificates Orchestrate — Unified search, time-series, geolocation, graph database and API Screenhero — Screen sharing for collaboration in teams SendGrid — Email infrastructure as a service Stripe — Web and mobile payments, built for developers Travis CI — Continuous integration platform for open source and private projects

The main thing I want to write about is Atom. Its github very own text editor and It is way more sleek than sublime. It is also free so you don’t have that annoying window popping up asking you to purchase it. I will have to go into more detail about it later since I am still getting use to it but I have definately gotten rid of sublime in favor over Atom.
